# Lista de Tarefas
Simple Todo list system, built with ElectronJS.

>## Install: 
>__git clone https://github.com/stefanluks/TodoList---ElectronJS.git__ <br> __cd TodoList---ElectronJS__ <br> __npm install__ <br> __npm start__ <br>
